## RESTful API with Spring framework

![세웃](API.JPG)

본 저장소의 API는 클라이언트-서버 간 원활한 연동을 위해 Spring Framework로 개발된 RESTful API입니다. MyBatis를 통한 MySQL DB 연동을 통해 CRUD를 비롯한 다양한 데이터 연동 기능을 구현하였습니다.
